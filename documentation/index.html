📘 OnLearn – React Educational Courses Landing Page

Thank you for purchasing OnLearn UI! This project is a clean and responsive educational landing page built using Vite + React + Tailwind CSS.

📁 Folder Structure

onlearn-frontend/
├── public/
├── src/
│   ├── assets/
│   ├── components/
│   ├── pages/
│   └── App.jsx
├── package.json
└── vite.config.js

🚀 Getting Started

To run this project locally:

git clone git@github.com:thephotogenicbug/onlearn-frontend.git
cd onlearn-frontend
npm install
npm run dev

🌐 Live Demo

https://onlearn-frontend.vercel.app/
